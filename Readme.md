# MarketPlace API

## Description 

- Api Marketplace is a web based app.
- User have to register or login via email and password.
- user can add, delete or modify the api's.
- user can contact us.




### Website Link : https://marketplaceapi.netlify.app/


### Technical Stack




```
backend
   
        MongoDB
        Nodejs
         

frontend

        HTML, CSS, JAVASCRIPT
        React

```
